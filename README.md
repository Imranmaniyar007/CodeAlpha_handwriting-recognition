# Handwritten Character Recognition using CNN

This project recognizes handwritten digits using a Convolutional Neural Network trained on the MNIST dataset.

## Features
- Deep learning CNN model
- Real-time web app using Streamlit
- 98%+ accuracy

## Tech Stack
Python, TensorFlow, Streamlit, NumPy

## How to Run
pip install -r requirements.txt  
streamlit run app.py
